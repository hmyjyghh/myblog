    前置知识

    # 前置知识

    # 1. clientX, clientY

    # 2. scrollTop, scrollLeft

    • scrollTop：设置或获取当前上滚的距离，即上卷的距离;
    • scrollLeft：设置或获取当前左滚的距离，即左卷的距离;

    # 3. getBoundingClientRect()

    • bottom: 元素底边（包括border）到可视区最顶部的距离
    • left: 元素最左边（不包括border）到可视区最左边的距离
    • right: 元素最右边（包括border）到可视区最左边的距离
    • top: 元素顶边（不包括border）到可视区最顶部的距离
    • height: 元素的offsetHeight
    • width: 元素的offsetWidth
    • x: 元素左上角的x坐标
    • y: 元素左上角的y坐标

    # 4. 获取实际位置

    let { left, top } = this.$refs.preview.getBoundingClientRect()
let x = e.clientX - left,
    y = e.clientY - top

// 此时 x,y 就是当前鼠标在指定元素内的实际位置

    # 使用

    # 1. 常用事件

    • ondragstart 事件
    • ondragover 事件
    • ondrop 事件

    # 2. 要拖拽的元素

    1. 给要拖拽的元素绑定 draggable="true" 属性, 变成一个可拖拽的元素
    2. 绑定 ondragstart 事件

    vue 项目使用

    <template>
  <ul>
    <li draggable="true" @dragstart="dragStart" data-name="Text">文本</li>
  </ul>
</template>

    methods: {
  dragStart (e) {
    let componentName = e.target.getAttribute('data-name')
    let info = {
      name: componentName
    }
    e.dataTransfer.setData('info', JSON.stringify(info))
  }
}

// 通过 e.dataTransfer 传递需要的数据

    # 3. 可拖放区域

    在可放置元素的区域，绑定 ondragover 事件 和 ondrop 事件

    vue 项目使用

    <template>
  <!-- @dragover="dropOver" -->
  <div class="preview" @dragover.prevent @drop="drop">
  </div>
</template>

    methods: {
  // 此方法可以简写为:  @dragover.prevent
  dropOver (e) {
    e.preventDefault()
  },
  drop (e) {
    let info = JSON.parse(e.dataTransfer.getData('info'))
  },
}

// 通过 e.dataTransfer 传递需要的数据

    # 4. Vue 中事件参数

    如果我们在 @dragstart 中想传递一些参数，如下：@dragstart="dragstart(item)"，就会遇到一个问题：默认传递的 DragEvent 参数丢失了。此时，我们需要使用 Vue 的特殊变量来实现事件参数的传递：

    @dragstart="dragstart($event, item)"
