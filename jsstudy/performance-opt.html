    # 1. еүҚзҪ®зҹҘиҜҶ

    # 1. д»Һиҫ“е…ҘurlеҲ°йЎөйқўжңҖз»Ҳе‘ҲзҺ°еҸ‘з”ҹдәҶд»Җд№Ҳпјҹ

    • еүҚдёҖдёӘйЎөйқўеҚёиҪҪ
    • DNSи§ЈжһҗпјҢжҠҠеҹҹеҗҚи§ЈжһҗжҲҗIPпјҢжүҚиғҪи®ҝй—®дё»жңә(жңҚеҠЎеҷЁ)
    • TCPдёүж¬ЎжҸЎжүӢ
    • RequestиҜ·жұӮ
    • Responseе“Қеә”
    • HTMLгҖҒCSSи§Јжһҗ
    • onLoadеҠ иҪҪ, йЎөйқўз»ҳеҲ¶

    # 2. TCPдёүж¬ЎжҸЎжүӢ

    1. з¬¬вјҖж¬ЎжҸЎвјҝпјҡе®ўжҲ·з«ҜеҸ‘йҖҒиҝһжҺҘиҜ·жұӮжҠҘвҪӮж®өпјҢе°ҶSYNдҪҚзҪ®дёә1пјҢSequence Numberдёәxпјӣз„¶еҗҺпјҢе®ўжҲ·з«ҜиҝӣвјҠSYN_SENTзҠ¶жҖҒпјҢзӯүеҫ…жңҚеҠЎеҷЁзҡ„зЎ®и®Өпјӣ
    2. з¬¬вјҶж¬ЎжҸЎвјҝпјҡжңҚеҠЎеҷЁж”¶еҲ°е®ўжҲ·з«Ҝзҡ„SYNжҠҘвҪӮж®өпјҢйңҖиҰҒеҜ№иҝҷдёӘSYNжҠҘвҪӮж®өиҝӣвҫҸзЎ®и®ӨпјҢи®ҫзҪ®Acknowledgment Numberдёәx+1(Sequence Number+1)пјӣеҗҢж—¶пјҢвҫғвј°иҝҳиҰҒеҸ‘йҖҒSYNиҜ·жұӮдҝЎжҒҜпјҢе°ҶSYNдҪҚзҪ®дёә1пјҢSequence NumberдёәyпјӣжңҚеҠЎеҷЁз«Ҝе°ҶдёҠиҝ°зҡ„SYN+ACKжҠҘвҪӮж®өвјҖе№¶еҸ‘йҖҒз»ҷе®ўжҲ·з«ҜпјҢжӯӨж—¶жңҚеҠЎеҷЁиҝӣвјҠSYN_RCVDзҠ¶жҖҒпјӣ
    3. з¬¬дёүж¬ЎжҸЎвјҝпјҡе®ўжҲ·з«Ҝж”¶еҲ°жңҚеҠЎеҷЁзҡ„SYN+ACKжҠҘвҪӮж®өгҖӮз„¶еҗҺе°ҶAcknowledgment Numberи®ҫзҪ®дёәy+1пјҢеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒACKжҠҘвҪӮж®өпјҢиҝҷдёӘжҠҘвҪӮж®өеҸ‘йҖҒе®ҢжҜ•д»ҘеҗҺпјҢе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«ҜйғҪиҝӣвјҠESTABLISHEDзҠ¶жҖҒпјҢе®ҢжҲҗTCPдёүж¬ЎжҸЎвјҝгҖӮ

    # 3. TCPеӣӣж¬ЎжҢҘжүӢ

    1. з¬¬дёҖж¬ЎжҢҘжүӢжҳҜжөҸи§ҲеҷЁеҸ‘е®Ңж•°жҚ®еҗҺпјҢеҸ‘йҖҒFINиҜ·жұӮж–ӯејҖиҝһжҺҘпјҢClientз«Ҝиҝӣе…ҘFIN_WAIT_1зҠ¶жҖҒ
    2. з¬¬дәҢж¬ЎжҢҘжүӢжҳҜжңҚеҠЎеҷЁж”¶еҲ°FINжҠҘж–ҮпјҢиҝ”еӣһACKжҠҘж–Үж®өиЎЁзӨәеҗҢж„ҸпјҢClientз«Ҝиҝӣе…ҘFIN_WAIT_2зҠ¶жҖҒ
    3. з¬¬дёүж¬ЎжҢҘжүӢжҳҜжңҚеҠЎеҷЁеҸ‘йҖҒFINжҠҘж–Үж®өпјҢиҜ·жұӮе…ій—ӯиҝһжҺҘпјҢClientз«Ҝиҝӣе…ҘLAST_ACKзҠ¶жҖҒ
    4. з¬¬еӣӣж¬ЎжҢҘжүӢжҳҜжөҸи§ҲеҷЁж”¶еҲ°FINжҠҘж–Үж®өпјҢеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒACKжҠҘж–Үж®өпјҢClientз«Ҝиҝӣе…ҘTIME_WAITзҠ¶жҖҒгҖӮжңҚеҠЎеҷЁжҺҘж”¶еҲ°ACKжҠҘж–Үе…ій—ӯиҝһжҺҘпјҢжөҸи§ҲеҷЁзӯүеҫ…дёҖж®өж—¶й—ҙеҗҺпјҢиЎЁзӨәжңҚеҠЎеҷЁе·Іе…ій—ӯиҝһжҺҘпјҢд№ҹе…ій—ӯиҝһжҺҘгҖӮ

        }
      }
    });

        // Force any pending records to be dispatched.
