    双指针算法

    # 双指针算法

    # 1. 什么是双指针？

    双指针和快速排序、冒泡排序等具体算法不同。它更接近于一种思路，一种使用两个指针互相配合来存储节点以便于运算的技巧。

    双指针法适用于数组、链表等线性数据结构，常用的思路有：碰撞指针、滑动窗口、快慢指针等。

    在两数之和这个case中我们使用碰撞指针的方式来实现

    # 2. 两数之和

    题目描述：

    给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

    示例:

    给定 nums = [2, 7, 11, 15], target = 9

    因为 nums[0] + nums[1] = 2 + 7 = 9 所以返回 [0, 1]

    # 3. 代码实现1：

    var nums = [2, 1, 7, 11, 15], target = 9;
// var nums = [1,2,2,3], target = 4;

function twoSum(nums, target) {
  const sortedNums = Array.from(nums).sort((a, b) => a - b);
  let right = sortedNums.length - 1;
  let left = 0;
  while (true) {
    const l = sortedNums[left];
    const r = sortedNums[right];
    const currentSum = l + r;
    console.log(l, r)

    if (currentSum === target) {
      return [nums.indexOf(l), nums.lastIndexOf(r)];
    } else if (currentSum > target) {
      right--;
    } else {
      left++;
    }
  }
}

console.log(twoSum(nums, target)) //[0, 2]
    • 需要注意的是，由于JavaScript引用类型的特性，我们首先拷贝了nums，才使用Array.sort对拷贝数组进行排序。

    # 3. 代码实现2：

    var nums = [2, 1, 7, 11, 15], target = 9;
// var nums = [1,2,2,3], target = 4;

function anwser(arr, target) {
  let map = {}
  for (let i = 0; i < arr.length; i++) {
    map[arr[i]] = i
  }
  // console.log(map) // {2: 0, 7: 1, 11: 2, 15: 3}
  for (let i = 0; i < arr.length; i++) {
    var d = target - arr[i]
    console.log(d)
    if (map[d]) {
      return [i, map[d]]
    }
  }
  return new Error('404 not found')
}
